# Multilingual System Troubleshooting Guide

## Common Issues

### Language Switching
**Problem**: Language doesn't change
**Solutions**:
1. Clear browser cache
2. Check language preferences
3. Verify feature flags
4. Restart browser

### Translation Loading
**Problem**: Translations not loading
**Solutions**:
1. Check network connection
2. Verify translation exists
3. Clear translation cache
4. Contact support

### RTL Layout
**Problem**: Right-to-left text issues
**Solutions**:
1. Update browser
2. Check CSS RTL styles
3. Verify font loading
4. Test different browsers

### Font Issues
**Problem**: Urdu fonts not displaying
**Solutions**:
1. Check font CDN
2. Clear font cache
3. Verify fallback fonts
4. Update browser

## Performance Issues

### Slow Loading
- Enable caching
- Use CDN
- Optimize queries
- Implement lazy loading

### Memory Usage
- Configure cache limits
- Monitor memory usage
- Optimize font loading
- Clear unused resources

## Error Codes

### Translation Errors
- `TRANS_001`: Service unavailable
- `TRANS_002`: Invalid language
- `TRANS_003`: Content not found
- `TRANS_004`: Quality too low

### System Errors
- `SYS_001`: Database error
- `SYS_002`: Cache unavailable
- `SYS_003`: API failure
