# Translator Training Materials

## Urdu Translation Guidelines

### Technical Terminology
- **Algorithm** → **الگورتھم**
- **Database** → **ڈیٹابیس** 
- **Machine Learning** → **مشین لرننگ**
- **Programming** → **پروگرامنگ**
- **Software** → **سافٹ ویئر**

### Cultural Adaptations
- Use Pakistani cities: کراچی، لاہور، اسلام آباد
- Currency: PKR format
- Local companies: Careem, Daraz, Jazz

### Quality Standards
- Accuracy: >90%
- Cultural appropriateness: 100%
- Consistency: Use approved terminology
- Readability: Natural Urdu flow

## Translation Workflow

### Step 1: Content Analysis
- Read source content thoroughly
- Identify technical terms
- Note cultural references
- Assess complexity level

### Step 2: Translation
- Use AI assistance for initial draft
- Apply terminology consistently
- Adapt cultural references
- Maintain original meaning

### Step 3: Review
- Self-review for accuracy
- Check terminology consistency
- Verify cultural appropriateness
- Submit for peer review

### Step 4: Revision
- Address reviewer feedback
- Make necessary corrections
- Resubmit for approval
- Track quality metrics

## Best Practices

### Do's
✅ Use approved terminology
✅ Maintain consistent style
✅ Adapt cultural references
✅ Preserve technical accuracy
✅ Follow RTL formatting

### Don'ts
❌ Literal word-for-word translation
❌ Ignore cultural context
❌ Mix formal/informal styles
❌ Skip terminology checks
❌ Rush through reviews
