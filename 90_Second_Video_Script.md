# 90-Second NotebookLM Demo Video Instructions
## Physical AI & Humanoid Robotics Textbook Project

## Step 1: Prepare NotebookLM (5 minutes setup)
1. Go to **notebooklm.google.com**
2. Create new notebook
3. Upload the `NotebookLM_Input_Document.md` file
4. Add any additional project files (README, code samples)
5. Click **"Generate Audio Overview"**
6. Wait for AI podcast generation (2-3 minutes)

## Step 2: Screen Recording Setup
**Tools needed:**
- Screen recorder (OBS Studio, Loom, or built-in screen recording)
- Your project website open in browser
- GitHub repository ready
- Audio from NotebookLM ready to play

## Step 3: 90-Second Recording Timeline

### **[0-15 seconds] - Introduction with Audio**
**Action:** Start NotebookLM audio, show website homepage
**Screen:** Navigate to your deployed book website
**Audio:** Let NotebookLM introduce your project

### **[15-30 seconds] - Book Structure Demo**
**Action:** Browse through different modules/chapters
**Screen:** Show Module 1 (ROS 2), Module 2 (Gazebo), Module 3 (Isaac), Module 4 (VLA)
**Audio:** Continue NotebookLM discussion about content

### **[30-50 seconds] - RAG Chatbot Live Demo**
**Action:** Interact with embedded chatbot
**Screen:** 
- Click on chatbot interface
- Type question: "How does ROS 2 work with humanoid robots?"
- Show AI response in real-time
**Audio:** NotebookLM explaining AI integration

### **[50-70 seconds] - Advanced Features Showcase**
**Action:** Demonstrate key features quickly
**Screen:** 
- Show login/signup (Better-auth)
- Click personalization button
- Click Urdu translation button
- Show content changing dynamically
**Audio:** NotebookLM discussing personalization

### **[70-90 seconds] - Technical Stack & Conclusion**
**Action:** Quick GitHub tour and wrap-up
**Screen:** 
- Open GitHub repository
- Show code structure briefly
- Return to live website
**Audio:** NotebookLM conclusion about project impact
