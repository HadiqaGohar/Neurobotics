# Project Status Skill

This skill provides automated project status checking, progress tracking, and report generation for book development projects.

## Capabilities
- Analyze project structure and completeness
- Generate progress reports
- Check for missing files or incomplete sections
- Validate project dependencies
- Create status summaries
- Track milestone completion
- Identify bottlenecks and issues

## When to Use
This skill is automatically invoked when:
- Generating weekly progress reports
- Checking project health before releases
- Identifying incomplete sections or chapters
- Validating project structure integrity
- Creating stakeholder status updates
- Monitoring development velocity

## Context Triggers
- Mentions of "project status", "progress report", "project health"
- Status check requests
- Progress tracking needs
- Project analysis tasks
