

# **Summary generator skill**

This skill condenses long texts, articles, or documents into concise summaries while retaining the key points and essential information.

## **Usage**

Provide a text or document, and the skill will return a brief, clear summary highlighting the main ideas.

**Example:**

**User:**
"Summarize this text: 'Artificial intelligence is rapidly transforming industries by automating tasks, improving efficiency, and enabling new capabilities in data analysis, healthcare, and robotics.'"

**Assistant:** *(Invokes summary generator skill)*

**Skill Output:**

```
Summary: Artificial intelligence is transforming industries by automating tasks and enhancing capabilities in data analysis, healthcare, and robotics.
```
