# Update README Skill

This skill automates README updates, ensures documentation stays current, and maintains consistent documentation standards.

## Capabilities
- Generate comprehensive README files
- Update project status and progress
- Maintain consistent documentation format
- Include relevant badges and links
- Update table of contents automatically
- Sync documentation with project structure

## When to Use
This skill is automatically invoked when:
- Creating initial README for new book projects
- Updating documentation after major changes
- Standardizing README format across projects
- Generating project status reports
- Maintaining up-to-date contributor information

## Context Triggers
- Mentions of "update readme", "generate documentation", "project documentation"
- Working with README.md files
- Documentation maintenance tasks
- Project status updates
