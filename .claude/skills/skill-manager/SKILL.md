# Skill Manager

This meta-skill provides a centralized way to discover, list, and understand all available skills and their capabilities.

## Capabilities
- List all available skills in the skills directory
- Provide skill descriptions and usage examples
- Organize skills by category and purpose
- Show skill relationships and dependencies
- Generate skill usage reports
- Help users find the right skill for their task

## Available Skills Inventory

### Content Skills
- **format-content**: Format and validate markdown content

### Documentation Skills
- **update-readme**: Generate and maintain README files

### Workflow Skills
- **project-status**: Check project status and generate reports

### Utility Skills
- **agent-discovery**: Discover available Claude subagents

## When to Use
This skill is automatically invoked when:
- Finding the right skill for a specific task
- Getting an overview of all available automation
- Understanding skill capabilities and limitations
- Planning workflow automation strategies
- Onboarding new team members to available tools

## Context Triggers
- Mentions of "available skills", "list skills", "skill capabilities"
- Questions about automation options
- Skill discovery requests
- Workflow planning needs

## Integration with Agents
This skill manager works alongside the agent-discovery utility to provide a complete view of available automation capabilities, including both skills and subagents.
