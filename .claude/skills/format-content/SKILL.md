# Format Content Skill

This skill helps format markdown files, validate structure, and ensure consistent styling across book content.

## Capabilities
- Format markdown files with consistent styling
- Validate markdown structure and syntax
- Check for common formatting issues
- Standardize heading hierarchy
- Clean up whitespace and line breaks

## When to Use
This skill is automatically invoked when:
- Processing imported content from various sources
- Standardizing formatting across multiple chapters
- Validating markdown before publishing
- Ensuring consistent style guidelines
- Cleaning up messy markdown files

## Context Triggers
- Mentions of "format markdown", "clean up content", "validate markdown"
- Working with .md files that need formatting
- Content consistency issues
- Markdown structure problems
